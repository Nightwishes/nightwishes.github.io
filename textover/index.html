如果遇到文本需要多行省略并且有展开收起功能，之前实现使用了js，今天看到一个方案可以使用纯css实现，记录一下

多行隐藏

这样就能多行省略,然后要实现展开按钮固定在文字右下角并且文字环绕有几个关键点

  1. 按钮要浮动
  2. 使用文字div的伪类来浮动，并且浮动的高度要div高度减去按钮的高度,这样浮动的按钮使用clear清除浮动就会固定在右下角(clear CSS 属性指定一个元素是否必须移动 (清除浮动后) 到在它之前的浮动元素下面。clear 属性适用于浮动和非浮动元素。)
  3. 文字div:after伪类想要自适应高度必须包裹在flex或者grid下
  4. 英文字母不支持要添加word-break:break-all

使用css选择器checked代替展开收起

接着添加一个input radio 按钮换成label并且关联input radio来代替js的展开，收起

  1. 使用css选择器当Input被选中的使用(展开按钮)改变文本的最大行数,改变label伪类的content:’展开(收起)’

文字内容不够使用伪类遮挡按钮

最后一个问题就是当文字字数少不要省略和展开按钮的时候按钮需要隐藏

在文本div添加一个伪类来遮挡按钮(当文本底部有足够的空间的时候证明不需要展开按钮，此时伪类会跟在文字后面)没有足够空间伪类会被挤下去,设置父元素over:hidden
